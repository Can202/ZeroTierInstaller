# ZeroTierInstaller

Tutorial:

1st: Download [this](https://github.com/Can202/ZeroTierInstaller/releases/download/v0.2/zerotier.sh)

2nd: Activate it as executable

3rd: Run in your terminal

4th: Follow the instructions
